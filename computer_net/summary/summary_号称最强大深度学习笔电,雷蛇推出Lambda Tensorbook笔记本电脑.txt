标题: 号称最强大深度学习笔电,雷蛇推出Lambda Tensorbook笔记本电脑

总结: IT之家 4 月 13 日消息，深度学习公司 Lambda 与雷蛇合作，发布了新的 Lambda Tensorbook 笔记本电脑，号称是世界上为深度学习设计的最强大的笔记本电脑，可以使用 Linux 和 Lambda 的深度学习软件。这款笔记本电脑配合 Lambda GPU 云，为工程师提供了他们在本地创建、训练和测试深度学习模型所需的所有软件工具和计算性能。Lambda 于 2012 年成立，已迅速成为世界领先的研究和工程团队的深度学习基础设施供应商。数以千计的企业和组织使用 Lambda，包括：所有前五大科技公司，97% 的美国顶级研究大学，包括麻省理工学院和加州理工学院，以及美国国防部。这些团队使用 Lambda 的 GPU 集群、服务器、工作站和云实例来训练用于癌症检测、自动驾驶飞机、药物发现、自动驾驶汽车等的神经网络。Lambda 联合创始人兼首席执行官 Stephen Balaban 称：“大多数 ML 工程师没有专用的 GPU 笔记本电脑，这迫使他们在远程机器上使用共享资源，拖慢了他们的开发周期。当你被困在 SSH 进入一个远程服务器时，你没有任何本地数据或代码，甚至很难向同事演示你的模型。雷蛇 x Lambda Tensorbook 解决了这个问题。它预装了 PyTorch 和 TensorFlow，让你快速训练和演示你的模型：所有这些都来自本地 GUI 界面。不再需要 SSH！”新的 Tensorbook 笔记本预先配置了 Lambda 的完整软件环境，包括 Ubuntu Linux 和 Lambda Stack，用于随时随地训练大型工作负载。这款笔记本电脑采用了雷蛇的时尚、高性能硬件，搭载 NVIDIA RTX 3080，这是目前最强大的移动 GPU 之一，并完全兼容 TensorFlow、PyTorch、cuDNN、CUDA 以及其他 ML 框架和工具。IT之家了解到，新的 Tensorbook 笔记本今天可以在 lambdalabs.com 上订购，价格为 3499 美元（约 22288 元人民币）。具体规格如下：硬件15.6 英寸 2560x1440 165 Hz 显示屏NVIDIA RTX 3080 Max-Q GPU，16GB VRAM英特尔 i7-11800 处理器（8 核，2.3 GHz 至 4.6 GHz）64GB DDR4 内存2TB SSD 存储雷电 4、USB 3.2、HDMI 2.1 端口4.4 磅（约 1997.6 克）铝制一体成型机身1080p 网络摄像头软件Ubuntu Linux 20.04 LTS （微软 Windows 双启动可选）带有 PyTorch、TensorFlow、CUDA、cuDNN 和 NVIDIA 驱动程序的 Lambda 堆栈一年的 Lambda 工程支持号称是世界上为深度学习设计的最强大的笔记本电脑，可以使用 Linux 和 Lambda 的深度学习软件。举报/反馈
